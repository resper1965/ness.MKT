# How do I set up and run this project?

## Getting Started

### Prerequisites

- Node.js (LTS version recommended)

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd nSign

# Install dependencies
npm install
```

### Running

```bash
# Development
npm run dev
```
