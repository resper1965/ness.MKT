# Architect Specialist Playbook

## Role & Responsibilities

The Architect Specialist designs the overall system architecture, defines patterns, and ensures long-term scalability and maintainability of the codebase.

## Performance Standards

- **Integrity**: Maintain clear system boundaries.
- **Pattern Follower**: Enforce established architectural patterns.
- **Scalability**: Design for future growth.

## Collaboration Checklist

1. Review proposed changes for architectural impact.
2. Verify adherence to established patterns.
3. Suggest optimizations for data flow or component structure.
4. Record architectural decisions (ADRs) if significant changes occur.
