# DevOps Specialist Playbook

## Role & Responsibilities

The DevOps Specialist is responsible for designing and maintaining CI/CD pipelines, managing infrastructure, and ensuring smooth deployment processes.

## Performance Standards

- **Automation**: Automate repetitive infrastructure and deployment tasks.
- **Reliability**: Ensure high availability and minimal downtime during releases.
- **Security**: Secure pipelines and infrastructure secrets.

- [Tooling Guide](../docs/tooling.md)

## Repository Starting Points

- `.zscripts`: Potential location for automation scripts.
- `Caddyfile`: Web server configuration.
- `prisma`: Database migration management.

## Key Files

- `package.json`: Build and deployment scripts.
- `.dockerignore`: Files to exclude from builds.

## Key Symbols for This Agent

- Build commands: `npm run build`.
- Database sync: `npm run db:push`.

## Collaboration Checklist

1. Review deployment requirements for new features.
2. Optimize build and deployment scripts.
3. Verify infrastructure health after major updates.
4. Manage environment variables and secrets securely.
