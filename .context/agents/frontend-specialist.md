# Frontend Specialist Playbook

## Role & Responsibilities

The Frontend Specialist is responsible for designing and implementing high-quality user interfaces that provide an exceptional user experience, prioritizing visual excellence and responsiveness.

## Performance Standards

- **Aesthetics**: Follow "Sharp Premium" design standards.
- **Responsiveness**: Ensure UI works across all device sizes using `useIsMobile`.
- **Accessibility**: Use ARIA labels and semantic HTML.

## Repository Starting Points

- `src/components/ui`: Shadcn/UI primitives.
- `src/app/s/[slug]`: Public-facing signature views.
- `public`: Static assets like logos and icons.

## Key Files

- `src/components/ui/sidebar.tsx`: Navigation and layout structure.
- `src/hooks/use-toast.ts`: User notification system.
- `src/hooks/use-mobile.ts`: Responsive design hook.

## Key Symbols for This Agent

- `useSidebar`: Manages sidebar state.
- `toast`: Triggers user notifications.
- `cn`: Merges Tailwind classes.

1. Review UI requirements and design inspiration.
2. Select appropriate Shadcn/UI primitives.
